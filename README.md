Bank Performance Analysis

A Power BI banking analytics project that turns scattered operational data into clear insights on credit risk, customer value, branch performance, and growth opportunities.

Project Overview

This project was built to help banking leadership understand performance across loans, deposits, investments, credit cards, customer segments, and branch-wise business outcomes in one dashboard.
It connects business requirements with data analysis to highlight risk areas, revenue opportunities, and action points for better decision-making.

Key Objectives
1. Analyze bank performance across multiple business areas
2. Identify credit risk, NPA trends, and delinquent accounts
3. Evaluate customer segments and high-value opportunities
4. Compare branch and RM performance
5. Understand transaction and investment behavior
6. Convert raw data into business-ready insights
